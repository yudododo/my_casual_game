## Overview 👀

### My Casual Game
- 遊戲類型必須為 Endless Mode，例如: 不斷閃避天空中掉下來的石頭，閃過得分，直到被打中，看能撐多久得到幾分。
- 有計分系統
- 可循環遊玩
- 必須用到 LocalStorage 儲存遊戲資料如最高分、關卡進度等
- 必須用到 Canvas
- 必須用到 WebGL
